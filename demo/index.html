gumy-widget — local demo

This page loads the local embed.js. The chat launcher floats in the bottom-right corner — click it, then press the expand button to go full screen.

Pick who to chat with via the URL, e.g. ?character=taylor-swift&origin=https://gumy.ai. The origin must point at a running gumy.ai that serves the embed API (/api/embed/character + /api/embed/chat).
